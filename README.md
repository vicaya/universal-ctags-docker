Getting Started
===============

Install
-------

```
curl -LO https://raw.githubusercontent.com/vicaya/universal-ctags-docker/master/ctags.sh
chmod +x ctags.sh
./ctags.sh --version
```

Usage
-----

```
./ctags.sh --help
```
